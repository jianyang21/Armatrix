# Armatrix Team Page

##  Live Links
- Frontend: https://armatrix-one.vercel.app/team
- Backend: https://armatrix.onrender.com/team

##  Tech Stack
- Frontend: Next.js, React, Tailwind CSS
- Backend: FastAPI (Python)

##  Features
- Dynamic team page powered by REST API
- Add, update, delete team members
- Responsive design (mobile + desktop)
- Dark theme UI
- Fun and creative team data

##  Setup Instructions

### Backend
cd backend
pip install -r requirements.txt
uvicorn main:app --reload

### Frontend
cd frontend
npm install
npm run dev

##  Design Decisions
- Used in-memory DB for simplicity and speed
- Dark theme for modern UI
- Modular component structure for scalability
